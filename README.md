## 🌐 Hosting with GitHub Pages

1. Create a GitHub repository
2. Upload all project files
3. Go to:
   Settings → Pages
4. Under “Source”:
   - Select `Deploy from branch`
   - Choose `main`
   - Select `/ (root)`
5. Save changes

Your site will be live at:

```text
https://manivel-cyber-ai.github.io/
```
